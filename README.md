# TT Preloved

A premium one-page selling site for high-end preloved products.

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Edit details

- Main page: `app/page.tsx`
- Styling: `app/globals.css`
- Product data: edit the `featuredProducts` array in `app/page.tsx`

## Suggested next steps

- Replace placeholder products with real items
- Add product photos
